## 🚀 Sobre o Projeto

O **SAPA** é uma aplicação voltada ao **autoconhecimento**, permitindo que usuários registrem hábitos e emoções e transformem esses dados em **insights comportamentais úteis**.

> 💡 A proposta é simples: transformar dados subjetivos em decisões mais conscientes.

---

## ✨ Diferencial

- 🧠 Integra **Psicologia + Tecnologia**
- 📊 Gera **insights automáticos**
- 🔍 Identifica **padrões comportamentais**
- 📈 Ajuda na **tomada de decisão**

---

## 🎯 Objetivos do MVP

| Objetivo | Descrição |
|--------|--------|
| 📊 Coleta | Registro de hábitos e emoções |
| 📈 Visualização | Interface simples e intuitiva |
| 🔍 Análise | Identificação de padrões |
| 💡 Validação | Testar impacto no autoconhecimento |
